This toolbox consists of a series of Linux command for discovery of microproteins.

Steps for using the toolbox

1) Prepare the data and install the software in software folder on Linux platform

2) Edit the file micrprotein_discovery_workflow.txt using the file name used in analysis

3) Run the command one by one on Linux platform

4) The results file will be in the result folder including known_ident.txt and novel_ident.txt
